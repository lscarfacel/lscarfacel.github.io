Chapter 31. Getting the current date and time

JavaScript Date objects represent a single moment in time in a platform-independent format. Date objects contain a Number that represents milliseconds since 1 January 1970 UTC.

var rightNow = new Date();
var theDay = rightNow.getDay();

Chapter 32. Extracting parts of the date and time

The getMonth() method returns the month in the specified date according to local time, as a zero-based value (where zero indicates the first month of the year).

var d = new Date();

var currentMonth = d.getMonth();

The getFullYear() method returns the year of the specified date according to local time.

var currYr = d.getFullYear();

The getHours() method returns the hour for the specified date, according to local time.

var currentHrs = d.getHours();

The getMinutes() method returns the minutes in the specified date according to local time.

var currMins = d.getMinutes();

The getSeconds() method returns the seconds in the specified date according to local time.

var currSecs = d.getSeconds()

Chapter 33. Specifying a date and time

var setDate = new Date("December 25, 2021").getTime() - new Date().getTime();
var daysTillChristmas = Math.floor(setDate / (1000 * 60 * 60 * 24));

Chapter 34. Changing elements of a date and time

The setFullYear() method sets the full year for a specified date according to local time. Returns new timestamp.

birthday.setFullYear(1991);

The setMonth() method sets the month for a specified date according to the currently set year.

birthday.setMonth(08);

The setDate() method changes the day of the month of a given Date instance, based on local time.

birthday.setDate(15);

The setHours() method sets the hours for a specified date according to local time, and returns the number of milliseconds since January 1, 1970 00:00:00 UTC until the time represented by the updated Date instance.

birthday.setHours(13);

setMinutes sets the minutes of an existing Date object without changing any other element.

birthday.setMinutes(18);

setSeconds sets the seconds of an existing Date object without changing any otherelement.

birthday.setSeconds(55);

setMilliseconds sets the milliseconds of an existing Date object without changing anyother element.

birthday.setMilliseconds(867);
