Chapter 12 if...else and else if statements

When using if statements you want something to happen when if doesn't test true. We use if...else for that.

Example

if (city === "Charlotte") {
 alert("Correct!");}
else {
 alert("Wrong city");}

else if

else if is used if all tests above have failed and you want to test another condition.

Example

 alert("Correct!");}
else if(city === "Moresville") {

Chapter 15 Arrays

An array is a variable that can have multiple values assigned to it

Example

You refer to each element by writing the array name countries in this case—followed by a number enclosed in square brackets. countries[0] is "USA"

Chapter 19 for loops: Flags, Booleans, array length, and loopus interruptus

A flag is just a variable that starts out with a default value that you give it.

var matchFound = "no"; If a match is found, the value of the flag is changed.

for (var i = 0; i <= 4; i++);
 if (countrieToGo === countries[i]) {

  alert("It's on the list, Lets Go!");}}
 if (countrieToGo === "no") {
