Chapter 58. The DOM

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web. You don't have to do anything special to begin using the DOM. You use the API directly in JavaScript from within what is called a script, a program run by a browser.

<body onload="console.log('Welcome to my Smarter Way To Learn JavaScript page!');"></body>

Chapter 59. The DOM: Parents and children

With the HTML DOM, you can navigate the node tree using node relationships.
You can use the following node properties to navigate between nodes with JavaScript:
parentNode
childNodes[nodenumber]
firstChild
lastChild
nextSibling
previousSibling
